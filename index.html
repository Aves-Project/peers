What is AVES?

AVES is a peer-to-peer network that relies on node connections to maintain network stability and security. By adding these peer nodes to your configuration, you help strengthen the network and improve your own connectivity.

How to Add Peer Nodes

Method 1: Using the static-nodes.json File

  1. Create a file named static-nodes.json in your node's root directory (/avesdir)
  2. Copy the JSON format from above and paste it into the file
  3. Save the file
  4. Restart your node or use the admin console to load the peers

Method 2: Using the Console (if available)

  1. Open your node's JavaScript console
  2. Use the admin.addPeer() function to add each node individually

Benefits of Adding Peer Nodes

  • Improved Connectivity: More peers means better network resilience
  • Faster Synchronization: Connect to trusted nodes for quicker syncing
  • Enhanced Security: A diverse set of peers improves network security
  • Better Performance: Reduce latency and improve transaction propagation
